# 실행 방법

### 1. 데이터 준비
- `public/sample[1-5]/` 디렉터리에 샘플 이미지를 준비합니다.
- `src/fonts/` 디렉터리에 학습에 사용된 폰트 *.ttf 파일들을 준비합니다.

### 2. Node.js 설치 (우분투)
```
curl -fsSL https://deb.nodesource.com/setup_16.x | sudo -E bash - &&\
sudo apt-get install -y nodejs
```

### 3. 패키지 설치
`npm install`

### 4. 서버 시작
`npm start`
